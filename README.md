# My Skills Website

This is a static website showcasing skills organized by category.

## Files
- `index.html` — main entry point for the website
- `styles.css` — styling for the website
- `bkg-1.png` — background image
- `my_skills.html` — alternate page version

## Local preview
1. Open `index.html` in your browser.
2. Or run a local server from this folder:
   ```sh
   python -m http.server 8000
   ```
   Then visit `http://localhost:8000`.

## Publish with GitHub Pages
1. Create a GitHub repository and push this folder.
2. In the repo settings, enable GitHub Pages from the `main` branch root.
3. Your site will be available at `https://<username>.github.io/<repo>/`.
